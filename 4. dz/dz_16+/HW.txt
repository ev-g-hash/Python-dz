Задание 1: Фильтрация положительных чисел
Напишите программу, которая принимает список чисел и использует filter(), чтобы вернуть новый список, содержащий только положительные числа.

Задание 2: Фильтрация строк по первой букве
Создайте список строк и используйте filter(), чтобы получить все строки, которые начинаются с буквы "A".

Задание 3: Фильтрация по длине слов
Напишите функцию, которая принимает список слов и возвращает новый список, содержащий только слова длиной больше 3 символов.

Задание 4: Фильтрация объектов по значению
Предположим, у вас есть список словарей, представляющих студентов с их именами и оценками. Напишите программу, которая использует filter(), чтобы вернуть список студентов с оценками выше 75.

Задание 5: Четные и нечетные числа
Создайте список чисел от 1 до 20. Используйте filter(), чтобы создать два списка: один для четных чисел, другой для нечетных.
